# Life Rhythm

یک وب اپ/موبایل فارسی برای مدیریت سبک زندگی چندنقشی با تمرکز روی تناوب کارها، فوریت، امتیاز روزانه و تداوم.

## امکانات نسخه فعلی

- onboarding برای انتخاب نقش های فعال
- داشبورد روزانه با امتیاز، پیشرفت و بیلبورد کارها
- فهرست امروز با رنگ بندی فوریت
- مدیریت نقش ها، ابعاد و افزودن تسک سریع
- تحلیل روند امتیاز و پیش نمایش امتیاز هر تسک
- ذخیره سازی محلی با `AsyncStorage`
- داده اولیه فارسی بر اساس سناریوی فایل نیازمندی

## معماری

- `App.tsx`: ورودی اپ
- `src/LifestyleApp.tsx`: UI اصلی و ناوبری داخلی
- `src/types`: مدل های دامنه
- `src/data`: seed data
- `src/utils`: تاریخ، تناوب و امتیازدهی
- `src/hooks`: state پایدار اپ
- `src/services`: لایه ذخیره سازی
- `src/components`: اجزای UI
